# Foodie Fix UI/UX Design Guide

## Design System

### Typography
- Primary Font: 'Inter', sans-serif
- Secondary Font: 'Poppins', sans-serif
- Font Sizes:
  - Headings: 32px, 24px, 20px
  - Body: 16px
  - Small Text: 14px
